#



### BADGES

Badges (ou distintivos) são emblemas visuais, majoritariamente digitais, que representam conquistas, habilidades, 
competências ou participação em uma atividade. Comumente usados em plataformas educacionais, corporativas e jogos, 
funcionam como reconhecimento não monetário, incentivando o engajamento e a gamificação. 
Eles funcionam como credenciais verificáveis e compartilháveis.
